Status zadania: Pełna poprawność
Uwagi: Możliwe krótsze rozwiązanie



Treść zadania:
Napisz program, który wczyta dwie liczby rzeczywiste z przedziału [-1024;1024] i wypisze ich różnicę.

Przykład:
Wejście:
-7
4.5
Wyjście:
-11.5
mała pomoc: input ( ), int ( ), float ( ), print ( )

Przykładowe rozwiązanie: 
a=input()
b=input()
a=float(a)
b=float(b)
c=a-b
print(c)
